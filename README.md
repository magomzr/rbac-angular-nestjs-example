Simple RBAC implementation using Angular, NestJS with permission strings, and PostgreSQL. This application uses the `action:resource` pattern, as a way to resolve users' access/authorization through roles (i.e. admin, editor, viewer, etc.).

Run with Podman using:

```bash
$ podman compose up --build
```
